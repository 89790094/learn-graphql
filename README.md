# learn-graphql

> 本教材服务器基于`express`,`express-graphql`,`graphql`.

## 开始

1. 克隆库:

```shell
git clone https://github.com/zhouyuexie/learn-graphql
```

2. 安装依赖:

```shell
cd learn-graphql && npm install
```

3. 运行:

```shell
npm start
```

现在打开你的浏览器输入`http://localhost:12580/graphql`,或者[点击这里](http://localhost:12580/graphql).
